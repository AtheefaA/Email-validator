# Email Validator (Python)

This project is a simple Python program that validates email
addresses using pattern matching techniques. It checks whether
an email follows the correct format.

## Features
- Validates email format
- Uses regular expressions
- Beginner-friendly Python code
- Quick and simple validation

## Technologies Used
- Python
- Regular Expressions (re module)

## How It Works
The program takes an email address as input and checks
whether it matches a valid email pattern.

## Example
Input: example@gmail.com  
Output: Valid Email
