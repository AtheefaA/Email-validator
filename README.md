# 📧 Email Validator

## 📌 Project Overview

The **Email Validator** is a Python application that checks whether an email address follows a valid format based on predefined validation rules. It helps users identify invalid email addresses and demonstrates the use of string manipulation, conditional statements, and pattern matching in Python.

---

## 🎯 Objectives

* Validate email addresses based on standard formatting rules.
* Improve understanding of Python string operations and validation techniques.
* Demonstrate practical use of conditional statements and regular expressions (if used).
* Provide quick feedback on email validity.

## 🚀 Getting Started

* Python 3.x installed on your system

1. Clone or download this repository.
2. Open the project folder.
3. Run the Python script:

```bash
python "Email Validator.py"
```

---

## 📖 Usage

1. Run the program.
2. Enter an email address when prompted.
3. The application checks the format.
4. A validation message is displayed indicating whether the email is valid or invalid.

---

## 📋 Validation Rules

The validator may check for:

* Presence of `@` symbol
* Valid username before `@`
* Valid domain name
* Presence of a domain extension (e.g., `.com`, `.org`, `.edu`)
* No invalid spaces or unsupported characters
